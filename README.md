# Knight Platformer Prototype

Platformer 2D sederhana yang dibuat menggunakan Godot 3.6 sebagai project pembelajaran game development.

## Features

* Movement
* Jump
* Roll
* Attack System
* Enemy HP
* Coin Collection
* Death Menu
* Win Screen
* Finish Level

## Engine

Godot 3.6
